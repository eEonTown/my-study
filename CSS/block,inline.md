# block, inline, inline-block

### block요소
> div, p, ul, il, h1~h6, form, table 등

##### 특징
- 사용가능한 최대 넓이를 사용
- 크기 지정 가능
- width : 100%,height 0으로 시작
- margin, padding 상, 하, 좌, 우 지정가능
- 레이아웃을 위한 요소

<br>

### inline요소
> span, a, b, string 등

##### 특징
- 필요한 만큼 넓이 사용
- 크기 지정 불가능
- width : 0, height : 0으로 시작
- 수평으로 배치됨
- margin, padding 좌, 우 지정가능
- 텍스트를 위한 용도

<br>

### inline-block요소
> img, input, button등

##### 특징
- 필요한 만큼 넓이 사용
- 크기 지정 가능
- 수평으로 배치됨
- margin, padding 상, 하, 좌, 우 지정가능
