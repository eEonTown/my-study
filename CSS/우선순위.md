# 우선순위

### !important
```css
div{color : red; !important;} 
```
### HTML인라인 선언 방식
```html
<div style="color : red;"></div>
```
### 아이디 선택자
```css
#a{color : red;}
```
### 클래스 선택자
```css
.a{color : red;}
```
### 태그 선택자
```css
div{color : red;} 
```
### 전체 선택자
```css
*{color : red;} 
```
### 상속
```css
body{color : red;} 
```
